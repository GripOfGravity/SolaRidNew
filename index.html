DATA ANALYTICS

SolaRid devices store data and display that data on the device dashbaord, allowing an in-depth data analysis.

Some of the data points the device collects are:

  • Insect species using AI technology
  • Daily temperature and humidity
  • Daily number of targeted insects killed

Each device can be customized with up to three sensors that can collect custom data points.

INFORMATION TRANSMISSION

For areas of 5-10 acres, a single master SolaRid device can be used. Areas larger than that can have any number of slave devices can be connected to the master device.

The basic flow of communication is:

  • Master device collects readings from slave units, if any are connected.
  • Master device sends data readings to the database server.
  • User dashboard displays data readings from the database server in real-time.

The user dashboard is accessible to anyone who has purchased a SolaRid device.

SOFTWARE APPLICATIONS

Each SolaRid device comes with access to a user dashboard that can be accessed via any modern web browser(Safari, Microsoft Edge, etc.).

Some of the main features of the user dashboard are:

  • Real-time updates from the sensors
  • Display graphs correlating temperature and insect densities
  • Configure and save device settings
  • View historical device readings data

The user dashboard is not required to use a SolaRid device, but is useful for monitoring insect populations and taking action.

TELEMATICS GPS

Track a SolaRid device's GPS coordinates and sensor readings by communicating over the cellular network. Remotely configure sensor settings through the user dashboard.

SENSING TECHNOLOGIES

Sensors are used to directly collect environmental conditions. The data collected from the sensors are then displayed on the web-accessible dashboard.

A closer look at the sensing technologies:

  • All SolaRid devices include sensors for temperature, humidity, and insect density per time period.
  • Up to three other sensors can be included in the SolaRid device to measure custom environmental condition variables.

Other AI technologies are being developed to supplement the sensors and provide a more accurate representation of the data.

DEVICE HARDWARE

SolaRid devices are durable and lightweight. They can easily be transported as insect populations move around.

The main features of a SolaRid device are:

  • Solar panels for device charging
  • Sensors for measures environmental conditions
  • LED lights and tone generators to attract incects
  • Interchangable chemical attractants that target specific insect species
